## Doc

TypeScript + Electron + Vite + Vue3 + 大漠插件

## ENV

node 12.18.3-x86

- choco install python visualcpp-build-tools -y

## Install

务必使用yarn安装、或者根据yarn.lock版本安装依赖

- yarn config set registry https://registry.npm.taobao.org

- yarn

- yarn rebuild

## Run

- yarn dev

## Build

- yarn build

- /dist/dm_electron_demo_setup_0.0.1.exe
